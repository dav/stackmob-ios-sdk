createMACHeaderForHttpMethod:path:

Creates the MAC header for OAuth2 authorization.

- (NSString *)createMACHeaderForHttpMethod:(NSString *)method path:(NSString *)path

Parameters

The HTTP verb to use, either POST,GET, PUT, or DELETE.

The REST path.

Return Value

A string that is set in signRequest: for the Authorization header.

Discussion
